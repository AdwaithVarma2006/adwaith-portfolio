## Overview

A minimal, typographically driven personal portfolio for **Adwaith Varma** — AI researcher, product scout, and builder. The site is intentionally lightweight: no frameworks, no build tools, just clean vanilla web tech.

## Features

- **Bold editorial layout** with large-scale typography using *Instrument Serif* and *Satoshi*
- **Dark / Light theme toggle** persisted via CSS
- **Custom cursor** for a polished desktop feel
- **X (Twitter) profile card** with live follower count
- **Social links** — GitHub, LinkedIn, Instagram, Email
- **GitHub Pages deployment** — zero-config hosting
- Fully **responsive** and accessible

## File Structure

```
adwaith-portfolio/
├── index.html       # Main HTML — structure & meta
├── base.css         # CSS reset / base tokens
├── style.css        # Component & layout styles
├── app.js           # Theme toggle, custom cursor, follower count
└── x-profile.jpg   # Profile image for X card
```

## Tech Stack

| Layer | Technology |
|-------|------------|
| Markup | HTML5 |
| Styling | CSS3 (custom properties, flexbox) |
| Scripting | Vanilla JavaScript (ES6+) |
| Fonts | Instrument Serif (Google Fonts), Satoshi (Fontshare) |
| Hosting | GitHub Pages |

## Getting Started

```bash
# Clone the repo
git clone https://github.com/theadwaithvarma/adwaith-portfolio.git

# Open locally
cd adwaith-portfolio
open index.html
```

No dependencies to install — just open `index.html` in any browser.

## Deployment

The site is automatically deployed via **GitHub Pages** from the `main` branch.

Live at: `https://theadwaithvarma.github.io/adwaith-portfolio`
